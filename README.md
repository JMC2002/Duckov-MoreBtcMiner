# 更多的比特币矿机 - More BTC Miner

## 功能

现在你可以制造更多比特币矿机了，默认增加两个，安装  
[ModSetting](https://steamcommunity.com/sharedfiles/filedetails/?id=3595729494)  
后可在设置里调整矿机的上限，调节至0即为无上限。

## 安全性

- 在启用本 MOD 前，请先手动取出原本矿机中的卡，如果未取出，可停用此MOD后在原本矿机中找回。
- 在卸载本 MOD 前，请先将所有矿机中的卡取出，如果未取出，可再次订阅后在原本矿机中找回。
- 在拆除矿机前，请先将矿机中的卡取出，如果未取出，可将矿机放回在拆除点完全一样的地方恢复找回。
- 本 MOD 不修改原本的任何存档内容，卸载后多余的矿机依旧存在，但是均指向同一个机器，。

## 安装

本 MOD 需要前置：  
• [JmcModLib](https://steamcommunity.com/sharedfiles/filedetails/?id=3613297900)  
• [Harmony](https://steamcommunity.com/sharedfiles/filedetails/?id=3589088839)  

请务必将这两个前置放在本 MOD 前加载。

如果需要游戏内修改矿机数量上限，请安装：  
• [ModSetting](https://steamcommunity.com/sharedfiles/filedetails/?id=3595729494)  
如果不安装，也可以在存档目录下的 `/JmcModLibConfig/MoreBtcMiner.json` 中手动修改。
